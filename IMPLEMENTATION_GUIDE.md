### 关键指标解读

#### ROI相关
- **ROI** = 销量 / 投入金额
- **CAC** = 投入金额 / 新客数
- **营销费率** = 营销费用 / 销量

#### 媒体效率
- **CPM** = 成本 / 曝光量 × 1000（越低越好）
- **CPC** = 成本 / 点击数（越低越好）
- **CTR** = 点击数 / 曝光量（越高越好）

#### 用户反馈
- **正面占比** = 正面评价 / 总评价（越高越好）
- **满意度** = 平均评分 / 5（越高越好）
